1. absolute units - px, cm, pt, mm, in
2. relative units - depend on something
  -> %, vw, vh, rem, em, vmin, vmax

  vw => viewport width
  vh => viewport height

  rem => root'em' , relative to the root element(<html>), 1rem = 16px.

  em => relative to the nearest ancestor

  % => relative to parent element ke dimensions parr
